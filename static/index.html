Definition
x Plaintext
y Ciphertext
α alphabet
a key component a
b key component b
a -1 mod inverse of key component a
len ( α ) length of alphabet
e ( x ) Encryption
d ( y ) Decryption
e ( x ) = y ≡ a ⋅ x + b mod len ( α )
d ( y ) = x ≡ a -1 ⋅ ( y − b ) mod len ( α )
Example
Given:
x = hello (7,4,11,11,14)
α = abcdefghijklmnopqrstuvwxyz
a = 7
b = 22
len ( α ) = 26
Encryption
e ( 7"h" ) ≡ 7 ⋅ 7"h" + 22 mod 26 = 19"t"
e ( 4"e" ) ≡ 7 ⋅ 4"e" + 22 mod 26 = 24"y"
e ( 11"l" ) ≡ 7 ⋅ 11"l" + 22 mod 26 = 21"v"
e ( 11"l" ) ≡ 7 ⋅ 11"l" + 22 mod 26 = 21"v"
e ( 14"o" ) ≡ 7 ⋅ 14"o" + 22 mod 26 = 16"q"
y = tyvvq (19,24,21,21,16)
Decryption
a -1 = 7 -1 = 15
d ( 19"t" ) ≡ 15 ⋅ ( 19"t" − 22 ) mod 26 = 7"h"
d ( 24"y" ) ≡ 15 ⋅ ( 24"y" − 22 ) mod 26 = 4"e"
d ( 21"v" ) ≡ 15 ⋅ ( 21"v" − 22 ) mod 26 = 11"l"
d ( 21"v" ) ≡ 15 ⋅ ( 21"v" − 22 ) mod 26 = 11"l"
d ( 16"q" ) ≡ 15 ⋅ ( 16"q" − 22 ) mod 26 = 14"o"
x = hello (7,4,11,11,14)
