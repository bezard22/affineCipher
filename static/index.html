Definition
x Plaintext
y Ciphertext
α alphabet
a key component a
b key component b
a -1 mod inverse of key component a
len ( α ) length of alphabet
e ( x ) Encryption
d ( y ) Decryption
e ( x ) = y ≡ a ⋅ x + b mod len ( α )
d ( y ) = x ≡ a -1 ⋅ ( y − b ) mod len ( α )
